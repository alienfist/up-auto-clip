## 🆕 多视角视频生成

从单个视频源生成多个不同风格和视角的视频：

```python
from main import AutoClip

# 初始化
auto_clip = AutoClip(video_path="your_video.mp4")

# 视频预处理
auto_clip.preprocess_video_segment()

# 生成所有10个视角的视频
results = auto_clip.generate_multiple_perspective_videos()

# 或者生成指定视角的视频
selected = ['emotional', 'entertaining', 'inspirational']
results = auto_clip.generate_multiple_perspective_videos(selected)
```

**演示脚本**: 运行 `python demo_multi_perspective.py` 查看完整演示。

**详细文档**: 查看 [MULTI_PERSPECTIVE_README.md](./MULTI_PERSPECTIVE_README.md) 获取完整使用指南。

## 🚀 快速开始

### 环境安装
```bash
# 创建虚拟环境
conda create -n up-auto-clip python=3.12

# 激活环境
conda activate up-auto-clip

# 安装依赖
pip install -r requirements.txt
```

### 配置环境变量
1. 把.env.example改名为.env
2. 并填写对应音乐和图片的API Key
3. 修改默认语言中文zh，英文en

### 使用方法
```bash
# 运行主程序
python main.py
```

# 背景音乐API配置说明
### Jamendo
- **注册地址**: https://developer.jamendo.com/
- **配置**: 需要获取Client ID
- **特点**: 音乐质量高，分类丰富

### Freesound
- **注册地址**: https://freesound.org/apiv2/
- **配置**: 需要获取API Key
- **特点**: 音效丰富，适合短音频


## 支持的视角类型

### 1. 默认通用 (default)
- **特点**: 适合各种类型视频的通用脚本，平衡考虑各种元素
- **适用场景**: 通用内容、综合性视频、不确定风格时的选择
- **配音风格**: 自然、平衡，适中的语调
- **字幕特色**: 清晰准确，易于理解

### 2. 情感共鸣 (emotional)
- **特点**: 突出情感元素、人物表情、温馨时刻
- **适用场景**: 家庭生活、感人故事、情感类内容
- **配音风格**: 温暖、感人，能够触动人心
- **字幕特色**: 简洁有力，突出情感关键词

### 3. 知识科普 (educational)
- **特点**: 突出知识点、技能展示、学习价值
- **适用场景**: 教程视频、技能分享、科普内容
- **配音风格**: 专业、清晰，便于理解
- **字幕特色**: 准确严谨，突出关键知识点

### 4. 娱乐搞笑 (entertaining)
- **特点**: 突出有趣、幽默、搞笑元素
- **适用场景**: 娱乐内容、搞笑视频、轻松话题
- **配音风格**: 活泼、幽默，增加娱乐效果
- **字幕特色**: 俏皮有趣，使用网络流行语

### 5. 励志激励 (inspirational)
- **特点**: 突出奋斗、成长、坚持等正能量
- **适用场景**: 励志故事、成功案例、正能量内容
- **配音风格**: 激昂、有力，充满正能量
- **字幕特色**: 鼓舞人心，传递积极信息

### 6. 美学艺术 (aesthetic)
- **特点**: 突出美感、艺术性、视觉冲击
- **适用场景**: 艺术作品、美景展示、文艺内容
- **配音风格**: 优雅、有韵律感
- **字幕特色**: 诗意文艺，有艺术气息

### 7. 热点话题 (trending)
- **特点**: 突出与热门话题、流行趋势相关的内容
- **适用场景**: 热点事件、流行文化、话题讨论
- **配音风格**: 紧跟潮流，使用流行语言
- **字幕特色**: 有话题性，容易引起转发

### 8. 生活方式 (lifestyle)
- **特点**: 突出日常生活场景、实用技巧、生活品质
- **适用场景**: 生活分享、实用技巧、日常记录
- **配音风格**: 亲切、自然，贴近生活
- **字幕特色**: 实用性强，贴近生活

### 9. 专业技能 (professional)
- **特点**: 重点展示专业技能、工作场景、职业发展
- **适用场景**: 职场技能、专业培训、行业分享
- **配音风格**: 专业、权威，具有说服力
- **字幕特色**: 专业术语准确，逻辑清晰

### 10. 故事叙述 (storytelling)
- **特点**: 以故事化的方式呈现内容，注重情节发展和叙事结构
- **适用场景**: 故事分享、经历叙述、情节性内容
- **配音风格**: 富有感染力，有节奏感
- **字幕特色**: 有故事性，引人入胜
